Contributors: Hailey Cho, Adia Lee, Christina Ng

## Overview

This project performs customer segmentation using clustering techniques and explores relationships among customer attributes through correlation analysis. It includes data preprocessing, exploratory analysis, dimensionality reduction, clustering evaluation, and visualization.

## Key Components

* **Data Loading & Cleaning**: Import raw customer data and handle missing values, outliers, and data types.
* **Feature Engineering**: Generate numerical and categorical features suitable for clustering.
* **Preprocessing Pipeline**: Scale numerical variables and one-hot encode categorical variables.
* **Clustering Methods**:

  * K-Means
  * Hierarchical Clustering
  * DBSCAN
* **Dimensionality Reduction**:

  * PCA for variance analysis and 2D visualization.
* **Model Evaluation**: Silhouette score, elbow method, and cluster stability checks.
* **Correlation Analysis**: Compute and visualize pairwise correlations.

## Files

* **Notebook**: Main workflow for clustering and correlation analysis.
* **Data**: Input dataset (not provided in repository).
* **Outputs**:

  * PCA plots
  * Cluster distribution charts
  * Correlation heatmaps
